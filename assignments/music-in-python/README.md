# 📘 Assignment: Music in Python

## 🎯 Objective

Build a simple music playlist analyzer in Python to practice working with lists, dictionaries, loops, and user input.

## 📝 Tasks

### 🛠️ Build a Playlist Data Structure

#### Description
Create a playlist represented as a list of dictionaries, where each dictionary stores song information such as title, artist, and duration in seconds.

#### Requirements
Completed program should:

- Store at least 5 songs in a list.
- Represent each song with keys: `title`, `artist`, and `duration`.
- Print all songs in a readable format.

### 🛠️ Add Playlist Analysis Features

#### Description
Write functions that analyze the playlist and allow a user to search or summarize songs.

#### Requirements
Completed program should:

- Calculate and print the total playlist duration in minutes and seconds.
- Find and print the longest song in the playlist.
- Let the user enter an artist name and print matching songs (or a message if none are found).
